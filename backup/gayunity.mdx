## Project Overview

**GayUnity** provides a safe, inclusive, and user-friendly platform for the LGBTQ+ community to connect. It features real-time messaging, premium membership options, secure authentication, and customizable user profilesâ€”all built for performance and scalability.

## My Role

I led the entire development lifecycle:

- **Frontend:** React (Vite) for speed and modularity.
- **Backend:** Node.js with Express for scalable APIs.
- **Real-time:** Socket.io for instant chat and notifications.
- **Payments:** Stripe integration for subscription-based premium features.
- **Database:** PostgreSQL for secure, relational data storage.
- **Deployment:** Configured for high availability and SSL-secured connections.
