## Project Overview

**Get Optimise** is an AI-powered SaaS application designed for teams to automate workflows, make data-driven decisions, and manage subscriptions seamlessly. The frontend is built with **Next.js (TypeScript)**, the backend uses **MySQL** with **Drizzle ORM** for type-safe access, and AI features are powered by **Python** services (Flask/FastAPI). **Stripe** handles secure subscription billing.

## My Role

I owned architecture, implementation, and deployment:

- <span className="font-semibold">Frontend:</span> Next.js (App Router) with TypeScript
  and reusable UI patterns.
- <span className="font-semibold">Backend:</span> MySQL + Drizzle ORM for schemas,
  migrations, and queries.
- <span className="font-semibold">AI Services:</span> Python microservices for inference,
  recommendations, and content generation.
- <span className="font-semibold">Auth & Payments:</span> Better Auth for RBAC +
  sessions; Stripe for subscriptions and invoices.
- <span className="font-semibold">Infra:</span> Containerized services with secure
  environment configuration and CI/CD.

## What I Optimised

- Hardened security with role-based access, input validation, and least-privilege API design.
- Reduced runtime errors via end-to-end type safety (TypeScript + Drizzle).
- Stabilized billing flows by reconciling Stripe events with auth roles in real time.
- Improved responsiveness using streaming UI and queue-backed AI jobs.

## Outcome

A robust, scalable **AI SaaS** foundation: teams automate repetitive work, get actionable insights, and manage subscriptions without friction. The typed stack and clear service boundaries enable **fast releases** and **confident iteration**.
